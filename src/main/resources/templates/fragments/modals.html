Creación de entidades

A orde para crear unha entidade (ordenador) é a seguinte:

  • Definir o procesador (opcional)
  • Definir a gpu (opcional)
  • Definir o modelo do pc (polo menos o fabricante e modelo son obrigatórios)
  • Agora, podes crear o ordenador e asignarlle o modelo

Isto é así dado que, para crear un ordenador, é preciso que exista un modelo asociado a el na base de datos.
